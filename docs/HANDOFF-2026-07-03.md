## 协作规则

**R1**: 执行前逐行脑内验代码
**R2**: 每回合更新 HANDOFF
**R3**: apply_patch 失败一次就换 Python 脚本
**R4**: 数据库查询给用户终端跑
**R5**: 数据同步命令给用户终端跑
**R6**: 禁止浪费 token — 数据库操作写命令让用户在终端执行
**R7**: 每次因子加减都归档到 FACTOR-AUDIT，不能改了又改
