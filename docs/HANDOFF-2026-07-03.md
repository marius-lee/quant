## 架构总览 — 7 层 Pipeline

```
data/ → 数据仓库 (SQLite, akshare 多源)
config/ → YAML 热加载
factor/ → 11 动态因子 + 5 基本面因子
alpha/ → 空壳 (逻辑在 pipeline/synth)
risk/ → 约束过滤 + 行业/市值中性化
optimizer/ → 组合构建 (资本自适应三档)
execution/ → 模拟执行 + 成本模型
monitor/ → 绩效报告 + Brinson 归因
web/ → Flask 仪表盘 (端口 8521)
```

## 协作规则

**R1**: 执行前逐行脑内运行 — 确认每个变量有来源、输出能到用户眼前。

**R2**: 每回合更新 `docs/HANDOFF-{date}.md`。

**R3**: 禁止浪费 token — `apply_patch` 失败一次就换 Python 脚本；先看缩进再改。

**R4**: 数据库查询给用户终端跑，不包在 Python 脚本里。

**R5**: 数据同步命令 (`sync_*`, `init_*`) 给用户终端跑。
