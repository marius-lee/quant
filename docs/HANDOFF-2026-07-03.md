## 架构总览 — 7 层 Pipeline

```
config.yaml → 结构参数 (佣金/滑点/因子阈值)
strategy_config.initial_capital → 种子本金 (写一次)
sim_trades.capital_after → 运行时资产 (每笔交易更新)

data/ → 仓库 (SQLite)
factor/ → 因子注册表驱动, 6 active
alpha/ → IC加权合成 (从 factor_registry 读权重)
risk/ → 约束过滤 + 中性化
optimizer/ → 组合构建
execution/ → 模拟执行
monitor/ → 绩效报告
web/ → Flask (8521)
```
