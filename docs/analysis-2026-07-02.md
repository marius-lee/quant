## 附加发现: ExecutionEngine.get_capital() 计算 bug

- **位置**: `execution/engine.py:89-93`
- `p[0]` 是 symbol (str), `p[1]` 是 SUM(shares), `p[2]` 是 avg_price
- 当前代码 `round(p[1],4) * max(0, p[0] - sell_map.get(p[0],0))`
  - `p[0] - number` → TypeError
  - 正确: `p[2] * max(0, p[1] - sell_map.get(p[0],0))`
- 此路径仅在 positions 非空时触发，若曾被调用会抛 TypeError
