# 动态基准设计

日期: 2026-06-05

## 问题

固定基准 (如沪深 300) 不代表策略的实际可投资机会集。
¥5,000 只能买 ≤¥50 的股票, 沪深 300 几乎全在 ¥50 以上。

## 方案

在每个调仓日, 用当日资金量对应的可买股票池构建等权基准:

```
回测第 1 天: 资金 ¥5,000
  → max_price = 5000/100 = ¥50
  → 基准池 = 当日所有 close ≤ 50 的股票等权平均

回测第 200 天: 资金 ¥50,000
  → max_price = 50000/100 = ¥500
  → 基准池 = 几乎全市场
```

## 实测结果

| | Strategy | 等权随机 | Alpha |
|------|:--:|:--:|:--:|
| Sharpe | -0.30 | +1.17 | -1.47 |
| 年化 | -13.9% | +40.4% | -54.4% |
| 最终 | ¥3,788 | ¥9,365 | -¥5,578 |

等权随机池翻倍了, 策略反而亏了 → 执行层有问题, 不是选股问题。

## 来源

- 回测黄金法则: 基准必须代表策略的可投资机会集
- WorldQuant/AQR/Two Sigma: 基准是自定义机会集, 不是第三方指数
