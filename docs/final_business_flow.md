# 最终业务操作流程 — 博采众家之长

日期: 2026-06-06  
来源: QMT + 聚宽 + Freqtrade + QuantConnect + 国际量化标准 + 个人实战

---

## 来源矩阵

| 平台/标准 | 采纳了什么 | 为什么不采纳其他的 |
|------|------|------|
| **QMT** | 盘前→盘中→盘后三段式 + 三种运行机制 | 不采纳: 本地绑定, 需一直开机 |
| **聚宽** | `run_daily` 定时函数 + `before_open`/`open`/`close` 时序 | 不采纳: 纯云端, 实盘通道已终止 |
| **Freqtrade** | 每K线评估 + `populate_indicators/entry/exit` 三回调 | 不采纳: 加密货币专用, 无A股概念板块 |
| **QuantConnect** | Universe→Alpha→Portfolio→Risk→Execution 管道 | 不采纳: 机构级重, 27.5万用户平台架构不适合单人 |
| **国际量化标准** | **Risk硬门控**(否决权,不是建议) + Scanner→Signal→Risk→Exec | 采纳核心 |
| **个人A股实战** | 盘后选股+次日执行 + 分散下单时间 + 14:00-14:55择时窗口 | 采纳核心 |
| **BigQuant** | AI可视化 + StockRanker | 不采纳: 闭源平台依赖 |
| **产业链传导(V5)** | 产业链阶段检测 + 板块热度排名 | 我们独创, 保留 |

## 最终流程

```
═══════════════════════════════════════════════════════════
                    初始化 (仅1次)
  • 加载产业链图谱 (V5)
  • 加载情绪周期模型
  • 设置风控参数
  • 注册定时任务
═══════════════════════════════════════════════════════════

═══════════════════════════════════════════════════════════
              Phase 1: 盘后选股 (15:00-20:00)
                    来源: 个人A股实战 + BigQuant

  ① 数据更新: 日线增量 + 财务数据 + 龙虎榜
  ② 产业链位置扫描 (V5): AI在哪? 半导体在哪? 新能源在哪?
  ③ 板块热度排名 (V4): 四维评分 → Top5热门板块
  ④ 涨停龙头检测 (V3): 在Top5板块内扫描个股
  ⑤ 生成明日候选池 (0-3只)
  ⑥ 发送通知: "明日候选: XX光电(光模块龙头, 评分0.91)"
═══════════════════════════════════════════════════════════

═══════════════════════════════════════════════════════════
              Phase 2: 盘前准备 (08:30-09:25)
                    来源: QMT + 聚宽

  ① 数据刷新: 隔夜新闻/政策/外盘
  ② 候选池复核: 是否有突发利空?
  ③ 负面清单过滤: ST/退市/减持/高管涉案
  ④ 情绪周期确认: 冰点? 复苏? 高潮?
  ⑤ 仓位计算 (V2): 根据情绪 × 个股得分 = 最终仓位
  ⑥ 下单准备: 挂涨停价限价单(排板)或市价单
═══════════════════════════════════════════════════════════

═══════════════════════════════════════════════════════════
              Phase 3: 盘中执行 (09:25-14:50)
                    来源: Freqtrade + QMT + 国际标准

  09:25  集合竞价后下单
  09:30~ 持续模式:
         ├─ Freqtrade模式: 每5分钟K线重新评估
         │  • 现有持仓: 是否触发止盈/止损?
         │  • 新候选: 是否有更优标的出现?
         │  • 如果新标的得分 > 持仓得分 × 1.3 → 换仓
         └─ 风险硬门控 (国际标准):
            • 单日亏损>5% → 熔断, 今日不再交易
            • 连续3笔亏损 → 暂停, 人工复核
            • 单票达到止损线 → 无条件执行, 风险经理有否决权
═══════════════════════════════════════════════════════════

═══════════════════════════════════════════════════════════
              Phase 4: 尾盘处理 (14:50-15:00)
                    来源: 个人A股实战(14:00-14:55最佳窗口)

  ① 未封板的股票 → 卖出(有盈利)
  ② 持有超过最大天数 → 强制卖出
  ③ 记录当日交易日志
═══════════════════════════════════════════════════════════

═══════════════════════════════════════════════════════════
              Phase 5: 盘后复盘 (15:00-16:00)
                    来源: QMT + 国际量化标准

  ① 绩效分析: 胜率/盈亏比/最大回撤
  ② 信号质量评估: 推荐的股票实际表现
  ③ 参数微调 (可选, 谨慎)
  ④ → 回到 Phase 1
═══════════════════════════════════════════════════════════
```

## 北极星数学(修正版)

```
不是6个月匀速复利 → 是抓住几次产业链大爆发

AI产业链传导: 概念期(轻仓试错) → 扩产期(全仓押注) → 制造期(换仓中游) → 退潮期(空仓)

一次产业链大爆发 = 全仓 + 龙头连板
  抓住2-3次大机会 × 每次50-100% = 3-8倍
  其余时间小赚小亏, 保住本金
  5000 × 3 × 3 × 2 = 90000 (4个月, 3次大机会)
  → 然后资金变大, 进入下一阶段, 可选择的股票变多
  → 循环加速
```
