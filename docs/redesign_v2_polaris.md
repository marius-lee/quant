# 量化系统重新设计 V2 — 北极星核心

日期: 2026-06-06  
目标: ¥5000 → ¥100万 (200倍, 6个月)  
来源: 游资打板战法 + 小资金做大实战体系 + 凯利极端复利数学

## 核心数学

```
200x / 6个月 / 125交易日

路径A: 8次交易, 每次 +95% (几乎不可能)
路径B: 28次交易, 每次 +20% (需要极高胜率)
路径C: 56次交易, 每次 +10% (需要极高频)

唯一现实路径: 高集中 + 高胜率 + 高频复利
```

## 北极星扫描器 (替代IC筛选)

```
不是: 80因子 → IC筛选 → ML模型 → 回测 → 推荐
而是: 涨停检测 → 龙头识别 → 资金确认 → 1-3只 → 全仓买入
```

### 扫描维度

| 维度 | 检测什么 | 参考来源 |
|------|------|------|
| 涨停模式 | 首板/二板/回封板/弱转强 | 北京炒家(首板), 赵老哥(二板), 陈小群(龙头) |
| 资金流向 | 龙虎榜席位 + 大单净买入 | 游资席位溢价 (呼家楼/六一中路/溧阳路) |
| 情绪周期 | 涨停家数/连板高度/炸板率 | 陈小群情绪周期体系 (已有) |
| 题材强度 | 政策催化 + 板块联动 | 主线题材识别 (AI/新能源/国企改革) |
| 量价关系 | 倍量突破 + 缩量回调 | 513战法, 烂板出妖股形态 |

### 排名逻辑

```python
def scan(store):
    candidates = []
    for strategy in [first_board, second_board, weak_to_strong, dragon_tiger]:
        candidates.extend(strategy.detect(store))
    
    # 按得分排序: 得分 = 涨停质量 × 题材强度 × 资金确认 × 情绪系数
    candidates.sort(key=lambda x: x['score'], reverse=True)
    
    # 情绪冰点 → 空仓或1只轻仓
    # 情绪高潮 → 全仓top 1-2
    mood = detect_mood()
    if mood == 'ice':
        return []  # 空仓
    elif mood == 'peak':
        return candidates[:1]  # 全仓1只
    else:
        return candidates[:2]  # 2只分散
```

## 和执行层的衔接

```
scanner     → 每天早上跑 → 输出 0-2 只 → 显示在Web界面
executor    → 用户确认/自动 → 清掉旧仓 → 全仓新推荐
monitor     → 盘中实时价格 → ATR移动止损 → 触发即卖
repeat      → 下次扫描时，如果新股得分>持仓股 → 换仓
```

## 和第一版的本质区别

```
V1: 一个"正常的"量化选股系统 → 年化15-20% → 100年才能到100万
V2: 一个"疯狂的"游资复利机器 → 每次全仓押注 → 8-56次正确操作即可
```

**V2 承认一个简单事实: ¥5000 到 ¥100 万不是靠分散风险完成的。是靠集中押注 + 高频复利。** 游资大佬们用的就是这个模式——只是他们不用Python写。

## 关键风险控制

```
1. 冰点空仓 — 市场没有机会时, 0仓位
2. 破线止损 — 5日线减半, 10日线清仓 (不是35%回撤)
3. 连续3次亏损 — 暂停交易, 反思 (游资铁律)
4. 永远不加杠杆, 不用生活费
```
