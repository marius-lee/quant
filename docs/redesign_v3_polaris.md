## Scanner: 三模式并联扫描

```python
# scanner.py — 参考 QMT-QuantLimit + KingPin

def scan(store) -> list[dict]:
    """09:00 扫描全市场 → 返回 0-2 只推荐"""
    
    mood = detect_mood()  # 已有: factor/market_mood.py
    
    if mood == 'ice':
        return []  # 冰点空仓 (赵老哥+陈小群铁律)
    
    candidates = []
    
    # 模式1: 首板检测 (北京炒家)
    first = detect_first_board(store)
    # 条件: 流通市值20-100亿, 封板<10:30, 换手5-15%, 板块≥3涨停
    
    # 模式2: 龙头识别 (陈小群)
    leader = detect_leader(store)
    # 条件: 连板≥2板, 板块跟风≥5只, 流通市值30-80亿
    
    # 模式3: 二板接力 (赵老哥)
    second = detect_second_board(store)
    # 条件: 首板爆量, 二板高开2-5%, 放量换手
    
    # 合并排名
    candidates = merge_and_rank(first + leader + second)
    
    # 按情绪周期决定持仓数
    if mood == 'peak':
        return candidates[:1]   # 高潮全仓1只
    else:
        return candidates[:2]   # 正常2只
```

## 和V1/V2的本质区别

```
V1: 通用量化系统 → 扫描多策略取共识 → 保守
V2: 游资战法量化 → 涨停+龙头+二板 → 激进
V3: 三大游资交集 + 两大开源项目验证 → 极简+铁血纪律

V3的核心不是"更多因子", 是"更少的错误":
  - 冰点不做 → 省掉50%的亏损
  - 不是龙头不做 → 省掉80%的杂毛亏损
  - 做错立即割 → 亏损永不扩大
  - 集中全仓 → 盈利最大化
```
