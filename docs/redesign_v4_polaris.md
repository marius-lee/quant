## 核心洞察

```
选股 = 个股信号 × 板块热度

个股信号 (已有的): 涨停检测 + 龙头识别 + 妖股检测
板块热度 (新增的): 涨停家数 + 龙头质量 + 资金流向 + 联动效应

只选 "个股强" 的 → 会买到孤军奋战的杂毛
只选 "板块热" 的 → 会买到跟风小弟
两者共振 → 买到主线龙头
```

## 决策流程

```
09:00 数据更新
  ├── ① theme.py: 扫描300+概念板块 → 四维评分 → 输出Top5热点主题
  ├── ② stock_scanner.py: 在Top5热点板块内, 扫描涨停/龙头/二板
  ├── ③ ranker.py: 个股信号 × 板块热度 → 综合排名 → top 0-2
  └── ④ executor.py: 买入 + 盘中监控 + 卖出
```
