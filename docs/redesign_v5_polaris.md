### 3. 叙事传播早期信号

| 信号类型 | 指标 | 来源 | 领先期 |
|------|------|------|:--:|
| **政策信号** | 政府工作报告关键词、产业政策文件 | 国务院/发改委官网 | 1-3月 |
| **机构信号** | 主题会议密度、研报覆盖频率 | 券商研报数据 | 2-4周 |
| **资金信号** | 北向资金/融资余额板块切换 | AKShare资金流向 | 1-2周 |
| **价格信号** | 板块成分股异常放量 + 涨停扩散 | 已有limit_up_pattern | 当日 |
| **叙事信号** | 新闻关键词共现网络变化 | 可选(NLP增强) | 1-4周 |

### 4. 产业链传导阶段检测

```
检测逻辑:
  IF 上游板块 涨幅/放量/资金流入 持续N天
  AND 上游板块 涨停家数开始扩散到同链其他板块
  AND 政策/新闻信号确认该主题
  THEN 标记该主题为 "活跃期"
  
  IF 主题活跃 + 中游板块开始出现资金异动
  THEN 该主题进入 "传导期" → 中游相关股票是下一阶段候选
  
  IF 主题活跃 + 下游板块开始被覆盖
  THEN 该主题进入 "扩散期" → 下游应用相关股票是候选
```
