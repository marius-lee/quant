# 中国券商金工因子回测流程标准汇总

> 综合华泰、中信建投、国泰君安、海通、广发、天风、招商、开源等券商金工研报

## 一、华泰证券（林晓明团队）

### 1.1 单因子测试系列（2017-2019）
- **报告标题**：《华泰单因子测试之资金流向因子》（多因子系列七，2018-05-17）
- **报告标题**：《华泰单因子测试之一致预期因子》（多因子系列九，2018-11）
- **报告标题**：《华泰单因子测试之海量技术因子》（多因子系列十一，2019-05-23）

#### 样本划分
- **时间窗口**：2014-01-03 至 2018-03-20（资金流向因子）；2009-01-31 至 2018-10-31（一致预期因子）
- **股票池**：全A股（剔除ST、PT、停牌股）
- **调仓频率**：日频截面（资金流向）/ 月频截面（一致预期）
- **换仓窗口**：每10个交易日换仓（分层回测）

#### 核心评价指标
| 指标 | 说明 |
|------|------|
| Rank IC均值 | 与下期收益的秩相关系数 |
| IC_IR | IC均值/IC标准差 |
| t值序列绝对值均值 | 回归法显著性 |
| \|t\|>2占比 | 显著性稳定性 |
| 分层单调性 | 分5/10层观察收益率单调 |

#### 数据处理
- 去极值：中位数法，D_M ± 5×MAD
- 中性化：对市值+行业回归取残差
- 标准化：Z-score
- 回归权重：流通市值平方根的WLS
- 交易成本：单边0.2%~0.3%；一致预期因子双边千四

### 1.2 人工智能系列（2017-2018）
- **报告标题**：《人工智能系列之五：人工智能选股之随机森林模型》（2017-08-31）
- **报告标题**：《人工智能系列之十：宏观周期指标应用于随机森林选股》（2018-03-20）

#### 7阶段滚动回测法
```
第1阶段：2011年训练 → 2011年测试
第2阶段：2012年训练 → 2012年测试
...
第7阶段：2017年训练 → 2017年测试
```
- **训练集长度**：6个月（兼顾风格切换敏感度和计算效率）
- **回测区间**：2011-02 至 2018-01（月频换仓）
- **股票池**：全A股（剔除ST、PT、停牌）

#### 样本外验证（风险调整动量因子，2018-10-23）
- **分界点**：2013年12月31日
- **样本内（In-Sample）**：2013-12-31之前数据，用于参数优化
- **样本外（Out-of-Sample）**：2013-12-31之后数据，用于验证
- **核心发现**：普通动量因子样本内有效参数中样本外仍有效仅7.92%；路径调整量因子提升至41.94%

## 二、中信建投（丁鲁明团队）

### 2.1 因子衰减专题
- **报告标题**：《深度专题67：因子衰减在多因子选股中的应用》（2019）
- **研究员**：丁鲁明、陈升锐

#### IC半衰期定义
月度IC第一次下降到当期IC一半或以下所用的期数。

**常见因子半衰期（全市场月频）**：
| 因子 | 类别 | 半衰期（月） |
|------|------|------|
| EP_TTM | 价值 | 3 |
| BP_LYR | 价值 | 3 |
| ROE_TTM | 质量 | 4 |
| Momentum_1m | 反转 | 1 |
| TurnoverAvg1M | 技术 | 3 |
| Volatility1M | 技术 | 3 |

#### 因子合成方法
- **推荐方法**：按因子半衰期分类 → 大类因子间等权 → 大类因子内按IC半衰期加权（H=H_Factor）
- **样本外效果**：近十年累计超额收益727%，年化23.52%，夏普2.08

#### 回归参数
- 滑动窗口N取12或24（影响不大，前5大IC权重占比~85%）
- 最大衰减期数通常选10期
- 中性化：流通市值对数 + 中信一级行业哑变量回归取残差

## 三、国泰君安

### 3.1 Alpha 191因子体系
- **报告标题**：《国泰君安191短周期因子系列》（2017-2018）
- **因子来源**：短周期价量数据，191个因子

#### 评估标准
| 维度 | 核心指标 | 判断标准 |
|------|---------|---------|
| 预测能力 | Rank IC均值 | 绝对值>=2%为有效 |
| 预测稳定性 | IR(IC_IR) | >0.3为稳健，>0.5为优秀 |
| 收益大小 | 因子收益率均值 | 正收益且显著 |
| 单调性 | 分层组合收益率 | 各层单调排列 |
| 交易成本 | 换手率 | 越低越好，通常<50%/年 |
| 风险控制 | 最大回撤、夏普比率 | 回撤小、夏普高 |

#### 分层回测参数
- **分层数**：5层或10层
- **调仓周期**：5-20个交易日（建议）
- **股票池**：中证500等中小盘
- **回归方法**：Fama-MacBeth回归，WLS以流通市值平方根为权重

#### 换手率参考标准
- <=50%/年：健康，适合实盘
- 50%-100%/年：审慎评估
- >100%/年：除非收益极高，否则不推荐

#### 实际案例
- 换手率因子：Rank IC均值-7.70%，IR -0.52
- 核心竞争力因子：月度IC 1.59%，ICIR 1.40

## 六、天风证券（吴先兴团队）

### 6.1 半衰IC加权专题
- **报告标题**：《专题报告十九：半衰IC加权在多因子选股中的应用》（2017-07-22）

#### 回测参数

| 参数 | 设定值 |
|------|--------|
| 回测区间 | 2010年1月 - 2017年6月30日 |
| 调仓频率 | 每月末调仓 |
| 股票池 | 全A股：上市>6月、非ST、非涨跌停/停牌 |
| 持仓数量 | 加权综合得分最大的100只 |
| 持仓方式 | 100只股票等权 |
| 交易成本 | 双边0.5%（买入0.2%+卖出0.3%） |
| 因子处理 | 除市值外，其他因子均行业+市值中性化；MAD去极值；Z-score标准化 |

#### 半衰IC加权核心参数
| 参数 | 取值 |
|------|------|
| 半衰期H | **H=2**（主报告推荐） |
| IC滚动窗口N | 过去24期（24个月） |
| 权重公式 | w_i = 2^(-(N-i+1)/H)，归一化 |

#### 加权方式对比
- 等权加权
- IC均值加权（过去24期IC均值）
- IC_IR加权（IC均值/IC标准差）
- 最优化复合IR加权
- **半衰IC加权（H=2）** —— 报告推荐

#### 表现
- 2017年1-6月超额全市场等权指数：+9.97%
- 相对最大回撤：-5.13%

## 九、跨券商共性总结

### 9.1 标准回测流程

```
因子构建 → 数据预处理(去极值+中性化+标准化) 
         → 单因子检验(IC+回归+分层) 
         → 多因子合成(等权/IC加权/半衰期加权/最优化)
         → 样本外验证(Walk-Forward/D分界点)
         → 绩效评估(IR/夏普/最大回撤/胜率/换手率)
```

### 9.2 常见参数范围汇总

| 参数维度 | 常见设定 |
|---------|---------|
| **回测区间长度** | 5-10年（2010年起常见） |
| **调仓频率** | 月频 > 双周频 > 周频 > 日频 |
| **股票池** | 全A（剔除ST/次新股/停牌），或沪深300/中证500/中证1000 |
| **分层数** | 5层（常用）或10层 |
| **交易成本** | 双边千二至千五（单边0.1%-0.3%） |
| **去极值** | 中位数MAD法（3-5倍MAD）或百分位法 |
| **中性化** | 市值+行业（中信一级）回归取残差 |
| **训练窗口** | 6个月（机器学习）至24个月（传统多因子） |
| **IC有效阈值** | |Rank IC| > 2% |
| **IR有效阈值** | > 0.3（稳健），> 0.5（优秀） |
| **换手率控制** | < 50%/年（健康），> 100%/年（高风险） |

### 9.3 样本外验证方法

| 方法 | 代表券商 | 说明 |
|------|---------|------|
| **固定分界点** | 华泰（2013-12-31） | 一刀切：IS参数优化 / OOS验证 |
| **7阶段滚动** | 华泰AI系列 | 每年重建模型，下一年测试 |
| **Walk-Forward** | 多家通用 | 250天训练+63天测试（A股推荐） |
| **三层隔离** | 行业共识 | 训练(60%)/验证(20%)/盲测(20%) |
| **CSCV** | 学术来源 | 组合对称交叉验证计算过拟合概率 |

### 9.4 常见偏差控制

| 偏差类型 | 处理方法 |
|---------|---------|
| 前视偏差 | 财务数据滞后（年报滞后4月、季报滞后1月）；shift(1)防未来数据 |
| 幸存者偏差 | 使用包含退市股票的完整数据集 |
| 过拟合 | 样本外验证、CSCV检验、参数稳健性分析 |
| 数据挖掘 | 基于经济学原理构建因子，而非纯数据试错 |
