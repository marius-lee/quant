## 问题背景

现有8个通过t检验的因子（IC均值2-5%），但只有2个的边际贡献IC显著。其余6个的增量被已选因子"吃掉"了——经典的多重共线性问题。

---

## 一、诊断：你的因子共线性有多严重？

### VIF（方差膨胀因子）

$$VIF_i = \frac{1}{1 - R_i^2}$$

其中 $R_i^2$ 是以第 $i$ 个因子对其他所有因子做OLS回归的拟合优度。

| VIF | 含义 | 建议 |
|:---:|------|------|
| <5 | 可接受 | 继续 |
| 5-10 | 中等共线性 | 需关注 |
| **≥10** | **严重共线性** | **必须处理** |

### 条件数（全局诊断）

$$CN = \sqrt{\frac{\lambda_{\max}}{\lambda_{\min}}}$$

| CN | 含义 |
|:---:|------|
| <10 | 无严重共线性 |
| 10-30 | 中等 |
| **≥30** | **严重** |

### A股因子典型VIF

| 因子对 | 典型VIF | 原因 |
|------|:---:|------|
| PE × PB | 高 | 同属估值类 |
| 资产负债率 × 权益比率 | 可高达**394** | 互为倒数 |
| Size × Turnover | 中等 | 大盘低换手/小盘高换手 |
| 不同周期动量 | 高 | 5日/10日/20日动量天然相关 |

**诊断流程**：先算条件数判断全局→再算VIF定位具体问题因子→用方差分解比例确认共线性结构。

> 来源：Hair et al.(1995); Belsley/Kuh/Welsch(1980); Barra CNE6 A股实证

## 二、五种正交化方法

### 方法1：Gram-Schmidt顺序正交

**算法**：
```
u₁ = f₁  (第一个因子保留)
For k = 2..K:
    u_k = f_k 对 [u₁..u_{k-1}] 做OLS回归的残差
```

**特点**：
- 等价于逐步回归取残差
- **顺序依赖严重**：排前面的因子信息完整保留，后面的被大幅修正
- 天风证券回测(2009-2017,7因子)：年化超额13.38%→18.83%，IR 1.71→2.66

**适用场景**：有明确因子优先级排序时

> 来源：海通证券冯佳睿/袁林青(2017.01.19)《选股因子的正交》

### 方法2：对称正交（Löwdin变换）⭐推荐

**算法**：
```python
M = F.T @ F                    # 因子协方差矩阵 K×K
eigenvalues, eigenvectors = np.linalg.eigh(M)
D_inv_sqrt = np.diag(1.0 / np.sqrt(eigenvalues))
S = eigenvectors @ D_inv_sqrt @ eigenvectors.T   # M^{-1/2}
F_orth = F @ S                  # 正交化后的因子
```

**特点**：
- **Carlson-Keller定理**：在所有正交变换中，对称正交使原始因子修改量最小
- 正交后因子与原始因子相关性>0.95（全部因子）
- 无顺序依赖，"民主"对待所有因子
- 天风证券回测：年化超额13.38%→**17.66%**，IR 1.71→**2.58**

**为什么推荐**：可解释性最强、不依赖顺序、计算最快（仅需截面数据K×K特征值分解）、实证表现优秀。在所有对比维度上均为最优或次优。

> 来源：天风证券吴先兴/张欣慰(2017.10.30)《因子正交全攻略》

### 方法3：PCA降维

**算法**：对因子相关矩阵做特征值分解，取前K个主成分作为新因子。

**特点**：
- 正交后因子**无法映射回原始因子**（经济含义丢失）
- 海通证券实证：PCA合成ICIR甚至**低于**等权法
- PCA最大化的是方差，不是预测能力(IC)
- 第一主成分与原始因子相关性可低至0.45（严重失真）

**适用场景**：因子数量很多(>50个)，不关心因子含义，纯预测导向

**不推荐**用于8因子场景。

> 来源：海通证券(2017.10.28)《因子降维1》；华泰金工(2019.01.04)

### 方法4：部分相关（Partial Correlation）/增量筛选

**精确公式**：
$$\rho_{XY·Z} = \frac{\rho_{XY} - \rho_{XZ} \cdot \rho_{YZ}}{\sqrt{(1-\rho_{XZ}^2)(1-\rho_{YZ}^2)}}$$

或等价于：分别将X和Y对Z做OLS回归，取残差的相关。

**与系统中marginal IC的关系**：
- A股常用的"中性化后IC"（只对因子X做中性化，不对收益Y做中性化）=**半偏相关(semi-partial)**
- 严格偏相关需要同时对X和Y做中性化=**边际IC**
- 两者数学等价但应用场景不同

**适用场景**：不需要完全正交化所有因子，只想检验新因子在已有因子基础上的增量信息时使用。

> 来源：海通证券(2017)；东方证券朱剑涛(2016)《Alpha预测》；FAJ(1990)

### 方法5：层次正交化

**算法**：按因子类别（估值/成长/动量/换手率/波动率等）分层，先在类内做正交，再跨类做正交。

**特点**：
- 目前A股研报中未作为独立正交方法被系统性实证对比
- 可以实现：每层内用对称正交，层间用回归残差法
- 可解释性优于全局正交化

**适用场景**：因子类别结构清晰，希望保留类别间天然低相关性时。

> 来源：东吴证券多因子体系；广发证券情景分层(2016)

## 四、正则化方法：Lasso/ElasticNet

### 精确公式

**Lasso** (L1):
$$\min_\beta \frac{1}{2N}\sum_i (y_i - x_i^T\beta)^2 + \lambda\sum_j |\beta_j|$$

**ElasticNet** (L1+L2):
$$\min_\beta \frac{1}{2N}\sum_i (y_i - x_i^T\beta)^2 + \lambda\left[\alpha\sum_j |\beta_j| + \frac{1-\alpha}{2}\sum_j \beta_j^2\right]$$

### A股实证效果

| 来源 | 方法 | 保留因子数 | IC/IR |
|------|------|:---:|------|
| 兴业(2020) | D_Net(修正EN) | 25.3/165 | IC=**0.135**, t=17.1 |
| 国盛(2019) | Adaptive Lasso | 自动 | IR=**2.79** vs ICIR法2.18 |
| 国盛(2019) | Group Lasso | 自动 | IR=**3.31**, 年化19.0% |

### 5493×120天可行性：✅ 完全可行（因子池≤200时~20分钟完成120截面）

**重要警示**：在弱信号环境下（IC普遍<0.015），Lasso可能灾难性地失效，Ridge更稳健。

> 来源：兴业证券徐寅(2020.02.22)；国盛证券刘富兵/丁一凡(2019.02.21)；修大成等

## 五、中性化 vs 正交化：本质区别

| 维度 | 中性化 | 正交化 |
|------|--------|--------|
| 作用对象 | 因子 vs 风险因子(行业/市值) | 因子 vs 其他Alpha因子 |
| 目的 | 剔除风险暴露，使因子纯净 | 消除因子间冗余信息 |
| 数学方法 | OLS回归取残差 | GS/对称正交/回归残差 |

**效果叠加**：华泰证券实证——ROE因子ICIR：原始0.91→行业中性化2.07→类别中性化**2.65**。两步效果叠加。

**A股最佳预处理顺序**：
```
去极值 → 中性化(行业+市值同时回归) → 标准化 → (多因子合成时)正交化
```

> 来源：华泰证券(2021.12)；海通证券；聚宽量化课堂；东北证券(2024)

## 七、针对你的场景（8因子，仅2个边际显著）推荐方案

### 推荐流程

```
第0步：VIF诊断
  - 对8个因子逐截面计算VIF
  - 条件数判断全局共线性
  - 对VIF≥10的因子对标记（如PE/PB、多周期动量）

第1步：合并明显重复的因子
  - 如果PE和PB都在，合并为"估值因子"（等权或IC加权）
  - 如果多周期动量都在，保留ICIR最高的一个
  - 目标：8因子→5-6因子

第2步：对称正交化
  - 对剩余因子做Löwdin对称正交
  - 验证正交后因子相关性<0.05
  - 重新计算每个正交因子的IC/ICIR

第3步：ICIR加权合成
  - 正交后因子 → ICIR加权 → 复合因子
  - 预期：复合因子ICIR提升30-50%

第4步（可选）：Lasso验证
  - 用Lasso在截面上自动选因子+赋权
  - 对比Lasso权重 vs ICIR权重
  - 如果Lasso把某些因子权重压到0→确认该因子确实冗余
```

### 为什么选对称正交而非Gram-Schmidt

| 对称正交 | Gram-Schmidt |
|---------|-------------|
| ✅ 无顺序依赖 | ❌ 顺序决定结果 |
| ✅ 所有因子可解释性>0.95 | ❌ 后排因子可解释性<0.80 |
| ✅ 计算最快 | ⚠️ 逐步回归较慢 |
| ✅ Carlson-Keller定理最优 | — |

**唯一例外**：如果你能确定因子的经济重要性排序（如"估值>动量>换手率"），GS固定顺序也可以——但对称正交不需要这个假设。

## 八、Python实现关键代码

```python
import numpy as np
from sklearn.linear_model import LinearRegression

# === VIF诊断 ===
def calc_vif(X):
    """X: (N_stocks, K_factors) 截面因子值"""
    K = X.shape[1]
    vif = np.zeros(K)
    for i in range(K):
        y = X[:, i]
        X_others = np.delete(X, i, axis=1)
        r2 = LinearRegression().fit(X_others, y).score(X_others, y)
        vif[i] = 1.0 / (1.0 - r2) if r2 < 1.0 else np.inf
    return vif

# === 对称正交 (Löwdin) ===
def symmetric_orthogonalize(F):
    """F: (N, K) 截面因子矩阵"""
    M = F.T @ F                              # K×K Gram矩阵
    eigvals, eigvecs = np.linalg.eigh(M)
    D_inv_sqrt = np.diag(1.0 / np.sqrt(np.maximum(eigvals, 1e-10)))
    S = eigvecs @ D_inv_sqrt @ eigvecs.T    # M^{-1/2}
    F_orth = F @ S                           # 正交化后因子
    # 可选：方差重缩放保持量纲
    scale = np.std(F, axis=0) / np.maximum(np.std(F_orth, axis=0), 1e-10)
    return F_orth * scale

# === Gram-Schmidt ===
def gram_schmidt(F):
    """F: (N, K)"""
    N, K = F.shape
    U = np.zeros_like(F)
    for i in range(K):
        u = F[:, i].copy()
        for j in range(i):
            u -= (u @ U[:, j]) / (U[:, j] @ U[:, j]) * U[:, j]
        U[:, i] = u / np.linalg.norm(u)
    return U

# === Partial Correlation / Marginal IC ===
def partial_corr(X, Y, Z):
    """X与Y在控制Z后的偏相关 (Z可以是多个控制变量)"""
    from sklearn.linear_model import LinearRegression
    resid_X = X - LinearRegression().fit(Z, X).predict(Z)
    resid_Y = Y - LinearRegression().fit(Z, Y).predict(Z)
    return np.corrcoef(resid_X, resid_Y)[0, 1]

# === Lasso筛选 ===
def lasso_select(X, y, alpha=0.01):
    """X: (N, K), y: (N,) 未来收益"""
    from sklearn.linear_model import LassoCV
    model = LassoCV(cv=5, max_iter=10000).fit(X, y)
    return model.coef_  # 非零系数=被选中的因子
```
