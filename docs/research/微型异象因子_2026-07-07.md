## 一、尾盘30min反转效应

**核心发现**：股市收盘前30分钟（14:30-15:00）存在显著反转效应。

### 精确公式（国泰君安2016）

```
DailyReverse = 0.5 × Reverse + 0.5 × Bias
Reverse = -1 × ln(Close / Close_14:30)    ← 尾盘涨越多→次日跌越多
Bias = (Close - VWAP_tail) / Close
DailyVolatility = 0.5 × std(5min_ret_tail) + 0.5 × ResidualVol
DailyMILLIQ = mean(|5min_ret| / 5min_amount) × 10^6
综合信号 = z(DailyReverse) + z(DailyVolatility) + z(DailyMILLIQ)
```

### IC数据

| 因子变体 | IC/IR | 来源 |
|---------|:---:|------|
| 三因子组合IC序列T值 | **6.21** | 国泰君安2016 |
| 黄金分割反转(10:00起算) | IC=-7.37%, ICIR=-4.04 | 华安2020 |
| 尾盘追涨(余弦相似度) | RankIC=-9.2%, IR=-4.04 | 开源2024 |
| 尾盘成交占比TTV | RankIC=5.45%, 周频6.54% | 信达2022 |
| 遗憾规避(逐笔小单) | 多空年化96.31%, 夏普8.77 | 国金2023 |

### 日内各时段IC分解（最精确数据）

| 时段 | IC均值 | 效应 |
|------|:---:|------|
| 13:00-13:30 | **-3.51%** | 🔥最强反转 |
| **14:30-15:00** | **-3.37%** | 🔥强反转 |
| 11:00-11:30 | -2.81% | 反转 |
| 隔夜 | +2.00% | 动量 |

### akshare：`stock_zh_a_hist_min_em(period="30")` 可获取30minK线。1min数据仅保留5日。

### 重叠风险：与20日反转因子高度重叠。建议正交化后仅保留增量信息。

> 来源：国泰君安(2016.12.26), 华安证券(2020.09.01), 开源证券(2024.06.23)

## 二、月份/季末效应

**核心发现**：A股存在显著的日历效应，但主要不是"1月效应"而是"春节效应"和"两会效应"。

### 精确统计

| 月份 | 历史平均超额 | 解释 |
|------|:---:|------|
| **2月** | **+2.1%** (小盘超额最强) | 春节后资金回流+两会预期 |
| 11月 | +1.3% | 机构年底排名冲刺 |
| **6月** | **-1.5%** | 半年度流动性紧张 |
| 12月 | -0.8% | 机构结算卖出 |
| 1月 | 不显著 | 中国没有显著"1月效应" |

### 量化方式

- **择时信号**：2月超配小盘/中证1000，6月低配 → 年化提升3-5%
- **截面哑变量**：在因子模型中加入 `is_february` 哑变量
- **不能做截面因子**：月份效应是时序的，不是截面的

### akshare：✅ 日频数据天然包含日期，零额外成本。

> 来源：中信建投年度策略, 多篇学术论文(JF中国特刊)

## 三、封闭式基金折价率（CEFD）情绪指标

**核心发现**：CEFD是A股最有效的单一情绪指标，领先市场约1个月。

### 精确公式

```
CEFD = (市场价格 - 单位净值) / 单位净值 × 100%
负值 = 折价(常见)，正值 = 溢价
从 akshare fund_etf_spot_em() 的 f402 列直接获取
```

### 预测力

| 方法 | 表现 | 来源 |
|------|:---:|------|
| 哑变量阈值法(>17%折价=买入) | 夏普1.98 | 招商2011 |
| CEFD情绪领先市场 | 约1个月 | 民生2022 |
| 科创板封闭基金轮动 | 超额26.8%(2024.11-2025.06) | 市场数据 |

### Baker-Wurgler中国版情绪指数中，CEFD因子载荷**0.913**，解释52.8%方差——最强单一指标。

### 关键相关系数

| 因子 | CEFD相关性 |
|------|:---:|
| **换手率** | **-0.42**（最高重叠风险） |
| **基金规模** | 解释71.89%截面方差 |
| 剩余期限 | -0.96（到期收敛） |

### 重叠风险处理：CEFD与换手率因子存在**高重叠**。建议对CEFD做规模中性化（回归log(NAV)取残差），并对换手率做去趋势处理（减5年均值，BW方法）。

### akshare：✅ `fund_etf_spot_em()` 的 `f402` 列直接返回折价率。注意传统封闭基金陆续到期转型。

> 来源：Baker & Wurgler(2006), 民生证券, 招商证券DUD体系

## 四、可转债隐含波动率-IV/HV差值

**核心发现**：可转债隐含波动率与正股历史波动率的差值（隐波差）是转债领域ICIR最高（-0.65）的估值因子。

### 精确公式

```
期权价格 C = (转债收盘价 - 纯债价值) / 转股比例
标的 S = 正股当日收盘价
行权价 K = 转债收盘价 / 转股比例
r = 1%（固定），T = 剩余期限（年化）

→ 二分法求解 BS模型 得 隐含波动率 IV

隐波差 = IV - RV(正股60日年化历史波动率)
```

### IC/IR数据

| 因子 | IC均值 | ICIR | IC胜率 | 多空年化 |
|------|:---:|:---:|:---:|:---:|
| **隐含波动率** | -0.08 | **-0.65** | 71.01% | 22.34% |
| **隐波差** | -0.07 | **-0.55** | 71.01% | 25.72% |
| 双低 | -0.07 | -0.42 | 65.22% | 23.56% |

> IV与隐波差相关系数**0.98**——绝对不要同时纳入多因子模型。

### 格兰杰因果检验：IV对正股指数有显著的领先预测力，最大影响T+2天，策略胜率最高74.93%（国海2026）。

### akshare：✅ `bond_zh_cov()` 获取转债-正股映射；`bond_zh_hs_cov_daily()` 获取转债日线；`stock_zh_a_hist()` 获取正股日线。纯债价值需自行贴现计算。

> 来源：华泰证券(2021.04), 西南证券(2023.12), 国海证券(2026.03)

## 五、ETF期权隐含偏度

**核心发现**：中国50ETF期权呈**右偏微笑**(与美股左偏相反)，SKEW对A股的预测力**强于VIX**。

### 关键发现：中国vs美股期权市场根本差异

| 指标 | 美国 | 中国 |
|------|:---:|:---:|
| 微笑形态 | 左偏(恐慌溢价) | **右偏**(散户乐观) |
| VIX与市场相关性 | -0.79 | **-0.12**(弱!) |
| SKEW与市场相关性 | -0.20 | **-0.66**(强!) |

### 核心因子公式

```
QVIX (中国版VIX): akshare index_option_50etf_qvix() 直接获取

IV偏斜: IV(OTM Put, K=0.95S) - IV(OTM Call, K=1.05S)
  高偏斜→后续下跌

成交量PCR: Put_Volume / Call_Volume
  A股是反向指标：高PCR(恐慌)→后续上涨
  年化择时17.04%, 夏普1.11(招商2024)

VRP(波动率风险溢价): QVIX² - RV_30²
  6个月期限预测力最强
```

### 因子重叠

| 重叠对象 | 程度 |
|---------|:---:|
| PCR vs 反转因子 | **高**（PCR极端值时=反转信号） |
| QVIX vs 历史波动率 | 极高 |
| SKEW vs 彩票型因子 | 中等 |

### akshare：✅ `index_option_50etf_qvix()` 直接获取QVIX；`option_value_analysis_em()` 获取各合约IV；需自行计算偏度/PCR/VRP。

> 来源：Yue et al.(2021), 招商证券(2024), 中信建投广义VIX(2023), 中泰期货(2025)

## 六、问询函/监管函事件

**核心发现**：交易所问询函发布后存在精确的三阶段漂移，75%的ST股前3年收过函。

### 精确三阶段漂移模型

| 阶段 | 窗口 | CAAR | 解释 |
|------|------|:---:|------|
| 事前 | t=-2~t=0 | **+0.78%** | 信息泄露/买预期 |
| **冲击** | **t=+1~t=+3** | **-0.55%** | 核心问询函冲击 |
| 收敛 | t=+4及以后 | ~0 | 回归正常 |

不同类型：重大资产重组类=**正向+5%**；财务报告类=**负向-3%**。

### ST预警力

| 指标 | 概率 |
|------|:---:|
| 收函后1年内被ST | **15.64%** |
| 收函后3年内被ST | ~19.94% |
| ST前3年收过函 | **75.51%** |
| 退市前3年收过函 | **74.83%** |

### 量化使用方式

- **负面清单复合因子**：问询函+财务异常+审计非标 → 排除（单一信号误杀率80%）
- **监管严重程度分级**：关注函 < 问询函 < 监管函 < 警示函 < 立案调查

### akshare：⚠️ `stock_notice_report()` + 标题关键词过滤"问询"/"监管"/"关注函"，无专用接口。

> 来源：陈运森等(2018)《金融研究》, 于孝建(2020), 兴业证券(2025)

## 七、质押比例变化

**核心发现**：质押比例与股价收益呈**非线性三次关系**，变化率(delta)的预测力强于存量水平。

### 精确非线性模型

$$
TobinQ = \beta_1 PledgeR + \beta_2 PledgeR^2 + \beta_3 PledgeR^3 + Controls
$$

| 质押区间 | 上升方向 | 机制 |
|---------|:---:|------|
| 0%-25% | **正面** | 激励效应，大股东有动力做好业绩 |
| 25%-70% | **负面** | 掏空动机，崩盘风险上升 |
| >70% | 再次正面 | 壳溢价、政府干预 |

熊市中高质押股票月收益低0.788%（年化-9.456%），但正常市场无显著差异——**状态依赖因子**。

### 推荐构建的增量因子

| 因子 | 公式 | 与已有pledge_ratio的增量 |
|------|------|---------|
| **delta_pledge** | (current-prev)/prev | 边际变化——上升=资金紧张→负面 |
| **plenew** | 本期新增质押/总股本 | 增量>存量的预测力 |
| **pledge_pressure** | 质押成本均价/当前价 | 股价维度，与比例正交 |
| **pledge_binary_30** | 1 if >30% else 0 | 非线性阈值(学术U型拐点) |

### akshare：✅ `stock_gpzy_pledge_ratio_em(date)` 直接返回全市场个股质押比例(%)、质押股数、质押市值、质押笔数。`stock_gpzy_pledge_ratio_detail_em()` 返回股东级明细含预估平仓线。

> 来源：汪先珍/马成虎(2022)《金融研究》, Li et al.(2023) JFDS, 鲍睿

## 八、员工持股计划(ESOP)

**核心发现**：公告日短期超额显著（+2.89%），但强烈的小市值暴露，沪深300内为负收益。

### 精确事件研究

| 窗口 | 超额收益 | 来源 |
|------|:---:|------|
| 公告日当天 | **+2.89%** | 章卫东(2016) |
| [-5,+5] | **+2.2%** | EFMA(2020) |
| 持有1月(中证1000) | **+37.4%** | 中信建投(2025) |
| 持有2月(中证500) | **+43.1%** | 中信建投(2025) |
| 持有1月(沪深300) | **-21.8%** | 中信建投(2025) |
| 持有12月以上 | 不显著 | 双刃剑效应 |

### 员工持股 vs 大股东增持的协同效应

大股东增持在"有员工持股"公司中CAR=**33.64%** vs 无员工持股的**10.12%**——员工持股提高了信息透明度，增强了增持信号可信度。

### akshare：❌ 无专门ESOP公告接口。可间接用`stock_gdfx_holding_analyse_em`(十大股东)和`stock_share_change_cninfo`(内部职工股)。完整数据需Tushare Pro或Wind。

> 来源：章卫东(2016), 方先明(2018), 中信建投(2025), 开源金工
