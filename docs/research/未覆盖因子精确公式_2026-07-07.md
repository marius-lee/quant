## 一、华泰大单净流入因子 mfd_sellord

**原始出处**：华泰证券林晓明《单因子测试之资金流向因子》（多因子系列之七）**2018.05.17**（非2022-2024系列）

**精确公式**：
```
mfd_sellord = -1 × (当日主力(超大单+大单)主动卖出委托单数量)
大单阈值: 20万~100万元 (固定绝对金额, 不分大小票)
超大单阈值: >100万元
主动卖出: 以买一价及以下价格成交的卖出委托
```

**akshare可用性**：`stock_individual_fund_flow_em()` 直接返回主力/超大单/大单净流入字段 → ✅可直接用

| 因子变体 | IC均值 | IR | 多空年化 | 夏普 |
|---------|:---:|:---:|:---:|:---:|
| mfd_sellord | 4.67% | 0.69 | 42.12% | **4.43** |
| mfd_volinflowrate_open_m | 5.51% | 0.78 | 14.71% | 3.12 |

**⚠️ 夏普误报**：用户声明夏普3.53实际是mfd_buyord(主力流入)的值，mfd_sellord真实夏普4.43

## 二、广发北向选股2.0 "踵事增华"

**原始出处**：广发证券周飞鹏/罗军/安宁宁《北向选股2.0：因子篇》**2024.05.24**

**精确公式**（三个子成分）：
```
1. 传统持仓占比 = 北向持股市值 / 流通市值 (存量, 风格正交后失效)
2. 特质变动 = ε from ΔHold ~ β₀ + β₁·Return_t-1 + β₂·ΔHold_t-1 (横截面回归残差)
3. 北向影响力 = 个股与相似北向行为股票的协同性 (精确公式未公开)
合成 = 等权或ICIR加权 (具体权重未公开)
```

**akshare可用性**：`stock_hsgt_individual_em(持股数量占A股百分比)` → ✅ 季频可用

**季频表现**：IC=5.1%, ICIR=2.7, 胜率89.3%, 调仓日为每季第5个交易日

**⚠️ 关键限制**：特质变动回归的完整自变量集、三个子成分的合成权重、以及北向影响力的精确公式均未在公开渠道披露，需PDF原文。

## 四、华泰OCFP（无2022-2024更新）

**原始出处**：华泰证券林晓明/陈烨《单因子测试之估值类因子》**2016.09.29**

**关键澄清**：华泰**未发布**OCFP的2022-2024更新。IBP和ENOA分别来自海通证券和学术研究，**不是**OCFP的改进版。

**精确公式**：
```
OCFP = 经营性现金流(TTM) / 总市值
TTM = 最近4个季度经营活动产生的现金流量净额求和
```

**OCFP vs FCFP对比**：

|---|:---:|:---:|
| ICIR | **0.526** (最高) | -0.03 |
| IC>0占比 | 69.85% | 48.9% |
| 因子收益率均值 | 0.168% | 0.042% |

华泰原话："在回归统计和IC值分析框架下，NCFP、FCFP效果不佳**可以删去**"

**季报滞后校正**：最佳方法=公告日(ann_date)对齐；保守法=年报滞后90天，季报滞后45天

**akshare**：`stock_xjll_em()` → "经营活动产生的现金流量净额"字段 ✅

## 五、西部TOI拉锯因子

**原始出处**：西部证券冯佳睿《因子手工作坊系列(1)》**2024.11.08**

**精确公式**：
```
Step 1: 定义拉锯日 = {t: r_night > 0 AND r_intra < r_night}
Step 2: TOI = PearsonCorr_{月度截面}(r_night - r_intra, Vol_intra/Vol_total)
   仅在拉锯日上计算
```

| 指标 | 数值 |
|------|:---:|
| IC均值 | **+0.035 (正向!)** |
| 年化ICIR | +2.75 |
| 胜率 | 83% |
| 多空年化 | 8.47% |

**IC正负号原理**：TOI捕捉"分歧后修复"（正向），华安昼夜因子捕捉"过度反应后反转"（负向）——方向完全相反是因为逻辑不同。

**akshare可用性**：日频OHLCV可算r_night/r_intra；但Vol_intra/Vol_total需要分钟数据拆分。无分钟数据时因子不可精确计算。

## 六、光大OCVP —— akshare不可算

**原始出处**：光大证券《见微知著：成交量占比高频因子解析》**2017.09**

**精确公式**：
```
OCVP = MA(Vol_集合竞价,t / Vol_总,t, 10)
OBCVP = 0.91 × OCVP + 0.09 × MA(Vol_收盘前5分钟 / Vol_总, 10)
```

**核心结论：akshare无法计算。** 没有任何免费数据源提供集合竞价成交量。日频OHLCV没有可用的近似替代（开盘跳空幅度与集合竞价量占比相关性不稳定）。

**替代方案**：VR因子 = Vol_上午前30分钟/Vol_下午前30分钟，IC>6%，IR>0.5，日频OHLCV可算 ✅

## 七、招商价变共振 —— ICIR=4.64待验证

**原始出处**：招商证券任瞳团队《青出于蓝系列之九》**2024.12.16**

**精确公式**（双维度4个因子）：
```
价格维度：CO_MAX/CO_MIN = rank(个股极值分钟的市场收益)/std(市场分歧度)
   取个股分钟收益率最高/最低各5分钟, 时序5日均值

成交量维度：PRE_AMT_MAX/PRE_AMT_MIN = (max/min极值分钟前1分钟成交量占比)/(日均占比)
   取个股分钟收益率最高/最低各30分钟, 领先1期

复合 = 0.25×等权四个子因子, 20日回溯, 5日半衰期指数衰减加权
```

**IC=6.32%是Pearson IC(非RankIC)，ICIR=4.64是普通ICIR(非RankICIR)**——这解释了其数值偏高。与EPA的RankICIR=4.75不可直接比较。

**akshare可用性**：技术上可行(1分钟数据)，但全市场约18亿行数据不现实。仅知乎个别复现了成交量子因子部分，无人完整复现。

**icIR评述**：按Grinold & Kahn标准(IR>1即优秀)，ICIR=4.64属异常级。直到独立对抗性复现完成前，应持**高度怀疑态度**。

## 八、东吴URet（信息分布不均）⚠️ 需要分钟数据

**原始出处**：东吴证券高子剑/庞格致《反应不足or反应过度？从信息分布到动量/反转》**2023.07.03**

**精确公式（关键：分钟成交量变异系数，非日频turnover！）**：
```
Step 1: 每天对每只股票计算 Z = std(分钟成交量) / mean(分钟成交量)
   = 分钟成交量的变异系数(CV), 衡量日内信息分布均匀度
   Z小=均匀分布(动量), Z大=集中分布(反转)

Step 2: 每月末, 过去20日按Z值从小到大排序, 等分5组(每组4天)
   计算每组的涨跌幅均值: RetPart1(Z最小) ~ RetPart5(Z最大)

Step 3: URet = RetPart5 = 过去20日中Z值最大4天的涨跌幅均值
   核心发现: 5组的年化ICIR呈严格单调递减(从正到负)
```

**不是HHI，不是Gini，不是日频turnover。** 报告使用变异系数(std/mean)作为信息分布代理变量。日内分钟成交量必须用分钟数据。

| 指标 | URet | 传统Ret20 | 提升 |
|------|:---:|:--:|:--:|
| IC均值 | -0.054 | -0.042 | +29% |
| 年化ICIR | -2.21 | -1.19 | +86% |
| 多空年化 | 21.18% | 15.41% | +37% |
| IR | 1.74 | 0.84 | +107% |
| 胜率 | 73.21% | 57.30% | +28% |

**akshare可用性**：❌ 需要分钟成交量数据计算CV，日频turnover无法替代。

## 九、华安加速换手 —— 日频可算，ICIR=4.29待验证

**原始出处**：华安证券吴正宇/严佳炜《加速换手因子："适逢其时"的换手奥秘》**2024.03.17**

**精确公式（关键：一阶差分，非比值！）**：
```
放量上涨日 = {Close_t > MA(Close,3) AND Volume_t > MA(Volume,3)}
日加速换手_t = Turn_t - Turn_{t-1}    (一阶差分, 非比值!)
因子值 = Σ_{t∈放量上涨日} (Turn_t - Turn_{t-1})  (20日窗口内条件求和)
```

| 指标 | 放量上涨日 | 大类复合 |
|------|:---:|:---:|
| RankIC | -10.5% | -10.2% |
| 年化ICIR | -4.29 | -4.57 |
| 月胜率 | 90% | 92% |
| 多空年化 | 35.1% | — |

**akshare可用性**：日频OHLCV完全可算 ✅。不需要分钟数据。

**与已有因子区别**：turnover_anomaly和turnover_rev_5d用的是除法（比值，衡量"水平"）；加速换手用**减法**（一阶差分，衡量"速度×情境"）——本质不同。

**ICIR评述**：与价变共振一样属异常级。小市值域主导（沪深300仅IC=-3.7%），中性化后ICIR降至-3.84。因子涉及择时信号选择（放量上涨日），存在多重比较偏误风险。独立复现前应持**高度怀疑态度**。
